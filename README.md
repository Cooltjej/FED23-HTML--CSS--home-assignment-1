### A webpage selling socks  

You are to create a simple website for a fictional company that sells socks. The website consists of four different HTML pages.  

------------------------------------------------------------------  

### General requirements:  

The website's name should be linked to the "home" page. The menu in the top right corner should have functional links to all pages. The focus of this assignment is on structure; all CSS development is optional.  
  
The "Contact" page contains a form - this does not need to send any emails or implement any other functionality. The form should have the correct elements and be structured correctly.  

------------------------------------------------------------------  

### Hygiene requirements:  

The code must be syntactically correct.  

The code must be semantically correct.  

The code must pass a W3C validation without warnings or errors.  

Your files should be well-structured and named appropriately.  

Use HTML comments to denote the different parts of the content (e.g., header, footer, etc.).
